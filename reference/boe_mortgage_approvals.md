# Download mortgage approvals for house purchase

Downloads the monthly count of mortgage approvals for house purchase, a
widely watched leading indicator of housing market activity. Available
from April 1993.

## Usage

``` r
boe_mortgage_approvals(
  from = "1993-04-01",
  to = Sys.Date(),
  seasonally_adjusted = TRUE,
  cache = TRUE
)
```

## Source

<https://www.bankofengland.co.uk/boeapps/database/>

## Arguments

- from:

  Date or character (YYYY-MM-DD). Start date. Defaults to
  `"1993-04-01"`.

- to:

  Date or character (YYYY-MM-DD). End date. Defaults to today.

- seasonally_adjusted:

  Logical. Return seasonally adjusted series (default `TRUE`) or
  non-seasonally adjusted (`FALSE`).

- cache:

  Logical. Use cached data if available (default `TRUE`).

## Value

A data frame with columns:

- date:

  Date. End of month.

- approvals:

  Numeric. Number of mortgage approvals.
